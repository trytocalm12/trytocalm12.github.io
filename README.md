# Audio Frontier Lab

> 探索声音的未来边界 | Exploring the Sonic Frontier

一个专注于音频前沿技术的 GitHub 主页项目，包含实时音频可视化演示和技术博客。

## 🌐 在线预览

访问 [audiofrontier.github.io](https://audiofrontier.github.io) 查看效果

## 🎵 特性

- ✨ 实时音频可视化（支持麦克风输入）
- 🎨 深色科技风格设计
- 📱 响应式布局
- 🚀 项目展示与文章列表
- 🛠️ 技术栈展示

## 🚀 快速开始

1. Fork 这个仓库
2. 修改 `README.md` 和 `index.html` 中的个人信息
3. 在仓库设置中启用 GitHub Pages
4. 访问 `https://你的用户名.github.io`

## 📝 自定义

### 修改项目
编辑 `index.html` 中的 `.projects-grid` 部分

### 修改文章
编辑 `index.html` 中的 `.article-list` 部分

### 修改技术栈
编辑 `index.html` 中的 `.tech-stack` 部分
